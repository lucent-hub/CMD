Server-Side Protection (Recommended)

Add to your .htaccess file:

# Block direct browser access to .lua files
<FilesMatch "\.lua$">
    SetEnvIf Referer "^$" direct_access
    SetEnvIf Referer ".*roblox.*" allowed_referrer
    Deny from env=direct_access
    Allow from env=allowed_referrer
</FilesMatch>
Lua Script Protection

Add to the beginning of your Lua script:

local function isLoadstringEnvironment()
    -- Environment detection logic
    return is_roblox or is_http or has_roblox_api
end

if isBrowserEnvironment() and not isLoadstringEnvironment() then
    -- Block unauthorized access
    return nil
end
